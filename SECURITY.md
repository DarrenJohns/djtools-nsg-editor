# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it responsibly:

1. **Open a GitHub Issue** — [Create an issue](https://github.com/DarrenJohns/djtools-nsg-editor/issues/new) with the label `security`
2. **Email** — Contact the maintainer directly via GitHub profile

Please include:
- A description of the vulnerability
- Steps to reproduce the issue
- Any potential impact

## Scope

This is a client-side, single-file HTML application with **no backend, no authentication, and no user data storage**. All data processing happens entirely in the browser.

## Response

Issues will be reviewed and addressed as soon as possible.
